# 🚀 AspiraSys AI-Driven Workshop Platform

A responsive and interactive landing page for the AspiraSys Workshop, guiding students through 5 hands-on tasks using AI tools like ChatGPT and DeepSite to build real-world digital projects.

---

## ✨ Key Features

* **Interactive Registration**
  Mobile-responsive modal with real-time validation and data stored using Local Storage.

* **Conditional Content Access**
  Certain features and sections become available based on user interaction or completion of prior steps.

* **Dynamic Progress Tracking**
  Visual feedback as users complete workshop tasks.

* **GSAP Animations**
  Smooth, modern animations for enhanced user experience.

* **Light / Dark Theme Toggle**
  User-controlled theme switching for better accessibility and UX.

* **Responsive FAQ Section**
  Accordion-style layout for common queries.

* **Mentor Support**
  Contact form with validation and success states, integrated with Google Apps Script to collect responses.

* **Modern UI/UX**
  Clean layout with custom SVG icons and smooth transitions.

---

## 🛠️ Tech Stack

* **HTML5 & CSS3** (Flexbox, Grid, Media Queries)
* **JavaScript (Vanilla)**
* **GSAP** (Animations)
* **Google Apps Script** (Form data handling)
* **Local Storage** (Client-side state management)
