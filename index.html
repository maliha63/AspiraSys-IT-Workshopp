TASK 01 Designing a Poster in Canva
1A — GUIDED TASK

Scenario: Create a poster for the 15th Anniversary of your College where Super Star Mr. Rajinikanth will be the Chief Guest on 25-Dec-2025 at 5 PM, Venue: Marudhar Kesari Jain College for Women - Vaniyambadi.

// Steps to Follow
  1. 1Open your browser and go to canva.com. Sign in or sign up using your Google account.
  2. 2Click on Create a design → search for Poster → select Poster (Portrait, A4 size).
  3. 3In the left sidebar Design tab, search: college celebration event free — Select a FREE template only (avoid crown icons).
  4. 4Click the title text and change it to: 15th College Anniversary
  5. 5Update the subtitle to: Chief Guest: Super Star Rajinikanth
  6. 6Update event details — Date: 25-12-2025 · Time: 5 PM · Venue: Marudhar Kesari Jain College for Women - Vaniyambadi
  7. 7Adjust colors, fonts, and elements to make it attractive. Then: Share → Download → PNG → Download
1B — TRY YOURSELF

Your department is organizing a Career Awareness Seminar on 15-08-2025 at 10 AM, at Central Auditorium, Main Campus - MKJCW Vaniyambadi. Speaker: Mukesh Ambani. Tagline: "Power Like You've Never Seen!"

Use only FREE templates. Follow the same steps as 1A. Add creative design touches: matching colors, icons, bold text.

Goal: Download & save both 1-A and 1-B poster images in one folder.

TASK 02 Creating a Logo & Branding Kit in Canva
2A — CREATING A LOGO

Create a professional logo for the brand ZACK'O using your initials ZK as the main symbol, with the tagline "Where Luxuries END" and a premium color palette.

// Steps to Follow
  1. 1Open Canva → Create a design → type and select Logo.
  2. 2In the Design tab, search: free personal brand logo — Select a FREE template that allows editing of two-word brand names and taglines.
  3. 3Type the brand name: ZACK'O. Customize using initials ZK as the main symbol.
  4. 4Add tagline: "Where Luxuries END" below or beside the brand name.
  5. 5Choose a premium color palette such as Black / Gold or Dark Blue / Rose Gold (avoid paid elements).
  6. 6Download: Share → Download → PNG (transparent background)
2B — TRY YOURSELF

Launch a fashion or beauty brand using your name initials. Reflect confidence, style, and luxury. Use a matching tagline, premium color palette (black & gold or navy & rose gold), and keep the logo clean and readable.

Goal: Design a personal logo reflecting your name, style, and future goals using initials, a tagline, and a premium color palette.

TASK 03 Getting Started with ChatGPT

ChatGPT is an AI chatbot by OpenAI that can understand and generate human-like text. Think of it as a smart assistant that can write, explain, answer, create, and solve — all in natural language.

// Login & Setup
  1. 1Go to chatgpt.com → Sign in or create an account with your Google email (free version).
  2. 2You'll see a chat screen where you can type a message — this is where you give a prompt.
// Sample Prompts to Try
  1. 3AGive me 5 interesting facts about Artificial Intelligence in simple language.
  2. 3BI have rice, eggs, onions, and some leftover chicken. Can you give me a fun and unusual recipe using just these ingredients?
  3. 3CI want to apply for an internship as a UI designer. Can you help me write a short and catchy cover letter that highlights my Canva skills and interest in design?
  4. 3DI want to start a YouTube channel about tech reviews. Can you suggest 10 unique video ideas, each with a catchy title?
3E — PORTFOLIO CONTENT ACTIVITY

In the next task you'll create a portfolio website. Use this prompt to generate your content. Fill in your own details before submitting.

Hi ChatGPT, I'm creating a personal portfolio website. Can you help me write content for the following sections in a simple, confident, and professional tone? I'm a student with basic tech skills. Here are the details: • Name: [Your Name] • Background: Final year B.Sc. Computer Science student • Skills: HTML, CSS, Canva, basic Python, teamwork, communication • Education: Jain College for Women, Vaniyambadi (2021–2024) • Languages: English, Tamil, Hindi • Interests: UI Design, blogging, exploring AI tools • Goals: To get my first IT job and build a strong career in tech Please give me content for: 1. About Me 2. Skills 3. Education 4. Languages 5. Interests 6. Career Goals

Copy all output and save it in Google Doc or Notepad for the next step.

TASK 04 Creating a Portfolio Website with DeepSite + Netlify
4A — BUILD WITH DEEPSITE

Use your ChatGPT content from Task 3E to build an elegant personal landing page portfolio using DeepSite.

// Building with DeepSite
  1. 1Go to www.deepsite.ai → Sign in with Google.
  2. 2In the chat screen, give this prompt (fill in your details from Task 3E): Create a simple and elegant personal landing page portfolio for myself. My name is [Your Name]. Include my name in a stylish text-based logo. Add menus: Home, About Me, Skills, Education, Languages, and Contact. In About Me: [Your bio from 3E] In Skills: [Your skills from 3E] In Education: [Your education details] In Languages: [Languages you speak] Make the site modern, elegant, and admirable with a clean design.
  3. 3Review the generated website and make sure your content from ChatGPT (Task 3E) is used across all sections.
  4. 4Click Copy on the left side to copy all the HTML code.
4B — DEPLOY ON NETLIFY
  1. 1Open any code editor (VS Code or Notepad) → paste the copied code → save as index.html
  2. 2Go to netlify.com → Sign up/log in with Google.
  3. 3Click Add new site → Deploy manually → drag and drop your index.html file.
  4. 4Netlify will host the page instantly and give you a link like: https://yourname.netlify.app
  5. 5Optional: Click Site settings → Change site name to get a custom subdomain like portfolio123.netlify.app

Save your Netlify link — you'll need it for submission. Preview your site on both desktop and mobile to ensure it looks great everywhere!

TASK 05 · OPTIONAL Creating a Fashion Website with Canva

Design and publish the ZACK'O Lifestyle Website — a luxury fashion/lifestyle brand using Canva's built-in website builder. Finish early? Try this for extra credit!

// Steps to Follow
  1. 1Open Canva → Search Website in the top bar → Browse Online Store or Lifestyle templates → Click Customize this template.
  2. 2Rename the project: ZACK'O Lifestyle Website
  3. 3Click Uploads → Upload your product and lifestyle images.
  4. 4Change site title to: ZACK'O – Where Luxuries END. Upload your logo to the top left.
  5. 5Use fonts: Playfair Display (headings) + Open Sans (body). Color palette: Black, White, Gold Accents.
  6. 6Add sections:
    • Home — Banner image with brand statement
    • About Us — Describe ZACK'O and your mission
    • Portfolio — Product images, lookbook, highlights
    • Contact — Email, phone, social media links
  7. 7Click Publish as Website → Choose Free domain → Get your public website link instantly!
